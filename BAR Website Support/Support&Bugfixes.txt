Common Issue Fixes

Issue #1: 
	The file cannot open in windows 10.

Fix:
	Click on the zip file and find the button that says extract all. Then open the newly created folder.
	
Issue #2:
	The BAR website is not updated.
	
Fix:
	Go to the BAR folder and find the latest update in the BAR/BAR%20Website/Website%20Releases/Beta%20Releases file path and download it.
